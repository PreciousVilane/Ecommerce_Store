# Ecommerce store

A simple e-commerce platform built with Django where vendors can add products and buyers can view them.

## Features

- User registration and authentication
- Vendor accounts to create stores, add products
- Product creation with images
- Static and media file handling
- tweets using third party API are made when vendors creates a new store or when they add new
products

## Installation

1. pip install -r requirements.txt
